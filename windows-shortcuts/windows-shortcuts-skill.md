# ⌨️ Windows 快捷键速查手册

> **记住一个快捷键 = 每天省 30 秒 × 365 天 = 3 小时/年。** 本手册按使用频率和场景分类，从你常用的 4 个开始，逐步扩展到 50+ 高频快捷键。

---

## 🎯 你最常用的 4 个（入门必记）

### Win + R — 运行

![Win+R 运行对话框](win-R.png)

**作用**：快速打开「运行」对话框，输入程序名或命令直接启动。

| 输入 | 打开 |
|------|------|
| `cmd` | 命令提示符 |
| `powershell` | PowerShell |
| `notepad` | 记事本 |
| `calc` | 计算器 |
| `mspaint` | 画图 |
| `control` | 控制面板 |
| `explorer` | 文件资源管理器 |
| `services.msc` | 服务管理器 |
| `regedit` | 注册表编辑器 |
| `cleanmgr` | 磁盘清理 |
| `msconfig` | 系统配置 |

> 💡 **技巧**：Win+R 是 Windows 最古老的快捷键之一（从 Windows 95 就有了），几乎所有系统工具都能通过它秒开。

---

### Win + X — 快捷菜单

![Win+X 快捷菜单](win-X.png)

**作用**：弹出系统快捷菜单（也叫「Power User Menu」），一键直达常用系统工具。

| 选项 | 说明 |
|------|------|
| 安装的应用(P) | 已安装应用列表 |
| 移动中心(B) | 显示亮度、音量、网络等 |
| 电源选项(O) | 睡眠/休眠/关机设置 |
| 事件查看器(V) | 系统日志查看 |
| 系统(Y) | 系统信息总览 |
| 设备管理器(M) | 硬件驱动管理 |
| 网络连接(W) | 网络适配器设置 |
| 磁盘管理(K) | 分区管理 |
| 计算机管理(G) | 综合管理工具 |
| Windows PowerShell(I) | 普通 PowerShell |
| Windows PowerShell (管理员)(A) | 管理员 PowerShell |
| 任务管理器(T) | 进程与性能监控 |
| 设置(N) | Windows 设置 |
| 文件资源管理器(E) | 资源管理器 |
| 搜索(S) | Windows 搜索 |
| 运行(R) | 等同于 Win+R |
| 关机或注销(U) | 关机/重启/注销子菜单 |
| 桌面(D) | 回到桌面 |

> 💡 **技巧**：右键点击任务栏的开始按钮也能打开这个菜单。这是 Windows 8 引入的「隐藏功能」，专门为高级用户设计。

### Win + Shift + S — 截屏 / 录屏

**作用**：调用 Windows 截图工具，支持区域选择、全屏、窗口等多种模式。

| 操作 | 效果 |
|------|------|
| 按下后拖拽鼠标 | 选择矩形区域截图 |
| 点击窗口 | 自动识别并截取整个窗口 |
| 按 Enter | 全屏截图 |
| 截图后自动复制到剪贴板 | 直接 Ctrl+V 粘贴到任意应用 |
| 截图后点通知弹窗 | 可用画笔标注、保存、分享 |

> 💡 **进阶**：截图后通知栏会停留几秒，点击可进入「截图与草图」进行标注（箭头、文字、马赛克）。
>
> 💡 **Win11 新增**：长按 Win+Shift+S 后可选择「屏幕录制」模式，录制选定区域的视频。

---

### Win + Shift + T — 提取文字

**作用**：Windows 11 的「文字提取」功能（Power Text），对屏幕上任意区域的文字进行 OCR 识别。

| 操作 | 效果 |
|------|------|
| 按下后选择区域 | 框选要识别的文字区域 |
| 识别完成 | 文字自动复制到剪贴板 |
| Ctrl+V | 粘贴提取的文字 |

> 💡 **适用场景**：
> - 图片里的文字 → 复制出来编辑
> - PDF 里无法选中的文字 → 提取
> - 视频字幕 → 实时提取
> - 浏览器里禁止复制的文字 → 绕过限制

> ⚠️ 此功能需要 **Windows 11 22H2 或更高版本**。

## 📌 常见误区

| ❌ 错误认知 | ✅ 正确理解 |
|------------|-----------|
| Win+R 只能打开 cmd | 几乎能打开任何程序/命令 |
| 截图只能用 QQ/微信 | Win+Shift+S 自带工具足够强大 |
| 只能用鼠标关窗口 | Alt+F4 是最快的方式 |
| 删除只能用 Del | Shift+Del 永久删除更快 |
| 切换窗口只能点任务栏 | Alt+Tab 效率高出 10 倍 |
| 锁屏只能 Win+L | Win+L 是唯一正确方式（别的不安全） |
